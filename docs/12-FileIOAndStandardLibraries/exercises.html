問題

#include <assert.h>

#define add(A, B) A + B

int main(void) {

    assert(6 == add(1, 2) * 2);

    return 0;
}
  • 上記のプログラムは、実行するとアサーション違反となる。main関数の中身を書き換えずに、期待通りの動作をするプログラムに書き換えよ。

問題

#include <stdio.h>

#define MAX(A, B) (A > B ? A : B)   

int main(void) {
    int a = 1;
    int m = MAX(0, a++);

    printf("m: %d\n", m);

    return 0;
}
